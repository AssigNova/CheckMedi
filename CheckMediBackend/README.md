# CheckMediBackend

A Node.js backend using Express.js and MongoDB (Mongoose).

## Setup

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Set your MongoDB URI in `.env` (default is local).
3. Start the server:
   ```powershell
   node index.js
   ```

## Features

- Express.js server
- MongoDB connection with Mongoose
- Sample route at `/`
